# dev-nodejs-web-api-expressjs-prisma-planetscale-simple

## Description
A POC for nodejs orm prisma connecting to local planetscale.

## Tech stack
- mysql
- planetscale
- typescript
    - express
    - prisma

## Docker stack
- node:latest
- mysql:8.0.32
- cloudmix/planetscale-local:latest

## To run
`sudo ./install.sh -u`
curl -i localhost/dogs

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
